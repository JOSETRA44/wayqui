El problema

Los préstamos informales generan conflictos

  • ✕Sin registro escrito, los montos se olvidan o se distorsionan.
  • ✕Los comprobantes de Yape/Plin se pierden en el chat.
  • ✕No hay historial de pagos parciales ni fechas de vencimiento.
  • ✕Cobrar resulta incómodo sin una herramienta neutral.
La solución

Wayqui es el registro neutral y seguro

  • ✓Historial inmutable con checksum criptográfico por préstamo.
  • ✓Adjunta comprobantes Yape/Plin directamente al pago.
  • ✓Acreedor confirma, rechaza o disputa cada transacción.
  • ✓Recordatorios por app, WhatsApp o correo electrónico.

Registro de Préstamos

Seguridad

Infraestructura de nivel bancario

Construido sobre la plataforma más confiable del mercado, con las mejores prácticas de seguridad financiera aplicadas en cada capa.

Verificación OTP de 6 dígitos

Confirmación de identidad por email en el registro. Sin contraseñas débiles.

Row Level Security (RLS)

Cada usuario accede únicamente a sus datos. Las políticas se aplican a nivel de base de datos, no de código.

Checksum SHA-256 por préstamo

Cada préstamo lleva un hash criptográfico. Cualquier modificación no autorizada es detectable.

Keychain / Keystore en dispositivo

Las sesiones JWT se almacenan en el enclave seguro del dispositivo (iOS Keychain, Android Keystore).

Tests de penetración automatizados

Suite de pruebas que verifica RLS con usuarios reales en cada release. La seguridad no se asume — se demuestra.
